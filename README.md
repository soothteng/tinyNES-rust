# tinyNES

**UNDER CONSTRUCTION**

## How to run

- Install rust environment
- move a nes file to directory and change its name to `test.nes`
- Finally, `cargo run`

## How to control

- Use Joystick on PC
- Use keyboard 
  | Control | Keyboard | 
   | ----------- | ---------- | 
    | Arrows | Arrows | 
    | A,B | a, s | 
    | Start | Enter | 
    | Select | Space |
